    Welcome to eXtaminal® Open Source

    eXtaminal® framework was born on 2005 and it is developed and maintained by Genectiva. Its purpose is to enable organisations to apply their functional and operation processes into IT. The commercial eXtaminal® was developed to be hard, it was created to exist in a insecure environment like Internet.

    As it is a modular framework, eXtaminal® can be easily incremented with extra module to enhance organization functional requirements.

    It was built to work with the Linux stack - Linux, Apache, MySQL and LDAP. In it's most recent commercial version, it integrates and run within a full Microsoft™ stack - Windows™, IIS™, SQL Server™ and Active Diretory™.

    The Open Source version works with CouchDB as an CouchApp making use of a commonly known User Interface Framework from Sencha - ExtJs.

    Real life implementations

    In its first version this framework was at work on various private and public organisations. One of the most demanding solutions was a Real Estate Franchising System embracing mixed technologies. As an integrated solution, eXtaminal® was configured to manage not only the Real Estate Information as well as to manage Domain Users full life-cycle, users email, network user folders, VPN connections as well as data sync between stores and main office. All this in an environment where Enterprise Linux and Windows desktop OS work together.

    Today eXtaminal® v2 is working mostly with governmental offices for tasks like the Study of Human Trafficking Phenomena within European Community. All its components are distributed and all communications are made by WebService Calls with military grade encryption. The framework evolved for a more technology embracing like Microsoft™ SQL Server and the authentication can now be done against LDAP or Active Dyrectory™
